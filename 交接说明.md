# Kaduosi Atelier V2 生产级品牌官网交接说明

## 1. 当前版本定位

当前版本已从单页询盘站升级为：

> 高端汽车内饰品牌官网 + SEO / SEM / GEO 长期增长底座

品牌方向：**Kaduosi Atelier - Premium Automotive Interior Programs**

## 2. 页面结构

| 页面 | 用途 |
|---|---|
| `index.html` | 品牌首页，承接品牌定位、三大产品方案和 RFQ |
| `catalog.html` | 产品目录页，承接产品类 SEO 与 SKU 询盘 |
| `oem.html` | OEM / private-label 落地页，承接 SEM 投放 |
| `resources.html` | FAQ / GEO 内容页，面向 AI 搜索和买家问题 |
| `contact.html` | 结构化 RFQ 转化页 |
| `404.html` | 错误页，避免流量丢失 |

## 3. 技术文件

| 文件 | 用途 |
|---|---|
| `assets/css/site.css` | 全站视觉系统 |
| `assets/js/site.js` | 产品数据、RFQ、语言基础和交互 |
| `robots.txt` | 搜索引擎抓取规则 |
| `sitemap.xml` | 站点地图 |
| `SEO_SEM_GEO_PLAN.md` | SEO / SEM / GEO 后续运营计划 |

## 4. SEO 已内置

- 每页独立 `title`
- 每页独立 `meta description`
- canonical URL
- Open Graph 标题
- JSON-LD 结构化数据：Organization / CollectionPage / Service / FAQPage / ContactPage
- sitemap.xml
- robots.txt

## 5. SEM 已内置

建议投放入口：

```text
/oem.html?utm_source=google&utm_medium=cpc&utm_campaign=custom_seat_cover_oem
/contact.html?utm_source=google&utm_medium=cpc&utm_campaign=sample_kit_rfq
/catalog.html?utm_source=google&utm_medium=cpc&utm_campaign=universal_seat_cover_wholesale
```

## 6. GEO / AI 搜索优化已内置

- 首页有 AI answer summary
- Resources 页面有 FAQPage JSON-LD
- 页面使用直接问答结构，方便 AI 搜索提取
- 公司实体、地区、类目和服务能力已结构化表达

1. 真实销售邮箱

3. 工厂、仓库、办公室照片

5. 车型安装实拍

7. Google Analytics / Google Search Console / Google Ads 转化追踪
